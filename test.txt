
Yes — but an important point first: there are very few frameworks designed only for data timeliness in the data quality literature. Most frameworks treat timeliness as one dimension among several. However, there are specialised frameworks and models that focus primarily on the timeliness or freshness of information.

Below are the most recognised frameworks specifically centred on timeliness/freshness.

1. Age of Information (AoI) Framework
The Age of Information (AoI) framework is one of the most widely recognised models specifically designed to measure data timeliness or freshness in information systems. AoI measures the time elapsed since the most recent update of information was generated, allowing systems to quantify how outdated the currently available data is. For example, in an SME logistics company tracking delivery vehicles through GPS sensors, AoI can measure how long it has been since the last location update was received; if the age becomes too large, the data is no longer timely enough for operational decisions. AoI therefore provides a direct metric for evaluating and optimising the timeliness of data updates in real-time systems (Kaul, Yates & Gruteser, 2012).

Reference
Kaul, S., Yates, R., & Gruteser, M. (2012). Real-Time Status: How Often Should One Update?
https://ieeexplore.ieee.org/document/6195689

2. Data Freshness Framework
Another framework that focuses on timeliness is the data freshness model, which evaluates how up-to-date information is relative to the time it was generated. This framework measures timeliness using latency between the real-world event and when the corresponding data becomes available for analysis or decision-making. For example, in a small retail SME, if point-of-sale transactions are only synchronised with the inventory system once per day, the data freshness framework would measure the delay and determine whether the information is sufficiently timely for stock management decisions (Golab, Johnson, Segev & Shoshani, 2014).

Reference
Golab, L., Johnson, T., Segev, A., & Shoshani, A. (2014). Data Freshness in Data Warehouses.
https://dl.acm.org/doi/10.1145/2596876.2596889
